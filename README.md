# Patolin Construction App

Sistema de gestión integral para empresa de construcción en Chile.

## Características

- ✅ Gestión de clientes y clientes potenciales
- ✅ Generación de facturas (formato chileno)
- ✅ Órdenes de compra
- ✅ Cotizaciones
- ✅ Integración con proveedores chilenos
- ✅ Comparación de precios de materiales
- ✅ Contabilidad bajo normas chilenas
- ✅ Sistema multi-usuario

## Instalación

```bash
# Instalar todas las dependencias
npm run install-all

# Ejecutar en modo desarrollo
npm run dev
```

## Estructura del Proyecto

```
patolin-construction-app/
├── client/          # Frontend React
├── server/          # Backend Node.js
├── database/        # Scripts de base de datos
└── docs/           # Documentación
```

## Tecnologías

- **Frontend**: React + TypeScript + Material-UI
- **Backend**: Node.js + Express + TypeScript
- **Base de datos**: PostgreSQL
- **Autenticación**: JWT
- **Moneda**: CLP (Pesos Chilenos)
